# E-Commerce-Website

## DesiCart
A simple front-end ecommerce demo built with HTML, CSS, and JavaScript.

## Overview
DesiCart is a student project that simulates an online shopping experience. It includes product browsing, category filtering, search, cart management, and a checkout flow with a responsive landing page design.

## Features
- Product grid with images and pricing
- Category filter buttons for clothing, jewellery, grocery, and electronics
- Live search for product names
- Shopping cart stored in `localStorage`
- Cart drawer with item quantities and total price
- Contact form with submission confirmation
- Responsive layout and festive UI styling

## Files
- `index.html` — main page structure
- `style.css` — page styles and responsive layout
- `script.js` — product filtering, cart logic, and interactivity
- `images/` — product images used in the storefront

## How to use
1. Open `index.html` in a browser.
2. Browse and search products.
3. Add items to the cart.
4. Open the cart drawer and checkout.

## Tech stack
- HTML
- CSS
- JavaScript

## Notes
This project is built as a front-end demo and does not connect to a real backend or payment gateway.
